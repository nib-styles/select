# select

Styles for a select.

## Installation

Component:
    
    component install nib-styles/select
    
Browserify:

    npm i --save @nib-styles/select
    
## Usage

    <div class="select">
        <select>
            <option>nib</option>
            <option selected>bupa</option>
            <option>hcf</option>
            <option>medibank</option>
        </select>
    </div>
